To view the prototype on a windows device, simply open the file in Adobe XD. On the top right-hand corner of the app, there will be a triangular play button. Clicking on this button will begin prototype mode.

To view the prototype on an iOS device, download the latest version of iTunes and connect the device to your laptop/desktop via USB.

To view the prototype on Android, save the file as a cloud document and login to the XD mobile app on the device. From there you can view the file on your mobile device.
